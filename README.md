# mini-contacts Project

Bu proje, Samsun Üniversitesi Yazılım Mühendisliği dersi kapsamında geliştirilen Python tabanlı bir rehber yönetim uygulamasıdır.

## 🚀 Versiyon 2 (V2) Yenilikleri
Ödev 5.2 kapsamında projenin V1 sürümü geliştirilerek aşağıdaki **3 somut görev** başarıyla tamamlanmıştır:

1. **Rehberden Kişi Silme:** Kullanıcı artık rehberdeki kayıtları isme göre silebilir. Silme işlemi öncesinde veri güvenliği için onay mekanizması eklenmiştir.
2. **Kişi Arama Özelliği:** Rehberde kayıtlı kişiler arasında isimle hızlıca arama yapma fonksiyonu getirilmiştir.
3. **Veri Kalıcılığı (JSON Entegrasyonu):** Rehberdeki veriler `rehber.json` dosyasına kaydedilmektedir. Bu sayede program kapatılıp açıldığında veriler kaybolmaz.

## 🛠 Kullanılan Teknolojiler
- Python 3
- JSON (Veri saklama için)

## 📋 Kurulum ve Kullanım
Projeyi çalıştırmak için terminale şu komutu yazabilirsiniz:
`python "Contacts V2.py"`
